# /scaffold-cli-tool

**Node.js CLI tool with commander, colored output, and file operations.**

Use this skill when building command-line tools and utilities with Node.js.

## What This Scaffold Provides

- 🖥️ Commander.js for CLI
- 🎨 Chalk for colored output
- 📁 File system operations
- 🔄 Async/await support
- 📦 npm publishing
- 🧪 Testing setup
- ⚡ Performance optimized

## Quick Start

\`\`\`bash
mkdir my-cli && cd my-cli
npm init -y
npm install commander chalk fs-extra
npm install -D typescript ts-node
npm start
\`\`\`
