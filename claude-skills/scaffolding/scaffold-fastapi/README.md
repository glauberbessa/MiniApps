# /scaffold-fastapi

**Python FastAPI backend with async database, authentication, and OpenAPI documentation.**

Use this skill when building a modern Python REST API backend.

## What This Scaffold Provides

- ⚡ FastAPI framework
- 🗄️ SQLAlchemy ORM + PostgreSQL
- 🔐 JWT authentication
- ✅ Pydantic validation
- 📚 Auto-generated OpenAPI docs
- 🧪 Pytest testing
- 🐳 Docker support
- 🔄 Async/await throughout

## Quick Start

\`\`\`bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
\`\`\`
