# /scaffold-rails

**Ruby on Rails REST API with PostgreSQL, JWT authentication, and ActiveRecord.**

Use this skill when building a Ruby on Rails API backend.

## What This Scaffold Provides

- 💎 Ruby on Rails framework
- 🗄️ PostgreSQL with ActiveRecord
- 🔐 JWT authentication
- 📡 JSON API responses
- 🧪 RSpec testing
- 📊 Database migrations
- 🐳 Docker support
- 🔄 Sidekiq for background jobs

## Quick Start

\`\`\`bash
rails new rails-api --api --database=postgresql
rails db:migrate
rails server
\`\`\`
