# /scaffold-django

**Django web framework with REST API, authentication, and admin panel.**

Use this skill when building a Python web application with Django.

## What This Scaffold Provides

- 🐍 Django framework
- 📡 Django REST Framework
- 🗄️ PostgreSQL database
- 🔐 Token authentication
- 📊 Admin interface
- 📧 Email configuration
- 🧪 Pytest testing
- 🐳 Docker support

## Quick Start

\`\`\`bash
django-admin startproject config .
python manage.py migrate
python manage.py runserver
\`\`\`
