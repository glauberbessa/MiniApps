# /scaffold-flutter

**Flutter mobile application with state management, navigation, and API integration.**

Use this skill when building cross-platform iOS and Android apps with Flutter.

## What This Scaffold Provides

- 📱 Flutter framework (Dart)
- 🧭 GoRouter navigation
- 📊 Riverpod state management
- 🎨 Material Design 3
- 🌐 HTTP API client
- 💾 Local storage (Hive)
- 🧪 Testing setup
- 🚀 Firebase integration

## Quick Start

\`\`\`bash
flutter create my_app
cd my_app
flutter pub add go_router riverpod dio
flutter run
