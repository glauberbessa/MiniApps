# /scaffold-electron

**Electron desktop application with React, local database, and auto-updates.**

Use this skill when building cross-platform desktop applications.

## What This Scaffold Provides

- 🖥️ Electron framework
- ⚛️ React frontend
- 🗄️ SQLite database
- 💬 IPC communication
- 🔄 Auto-updates
- 🔐 Security best practices
- 🧪 Testing setup

## Quick Start

\`\`\`bash
npx create-electron-app electron-app --typescript
npm install react react-dom better-sqlite3
npm start
\`\`\`
