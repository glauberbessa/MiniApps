# /scaffold-express

**Express.js REST API backend with database, authentication, and validation.**

Use this skill when building a Node.js REST API backend with Express.

## What This Scaffold Provides

- 🔷 Express.js server
- 🗄️ PostgreSQL + Prisma ORM
- 🔐 JWT authentication
- ✅ Zod validation
- 📊 OpenAPI documentation
- 🧪 Jest testing
- 🔒 Security headers
- 🚀 Docker support

## Quick Start

\`\`\`bash
npm install
cp .env.example .env
npm run db:push
npm run dev
\`\`\`
