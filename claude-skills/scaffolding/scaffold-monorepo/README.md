# /scaffold-monorepo

**Turborepo monorepo with shared packages, build caching, and workspace management.**

Use this skill when managing multiple related packages in a single repository.

## What This Scaffold Provides

- 📦 Turborepo for build caching
- 🔗 Workspace configuration (pnpm)
- 📚 Shared packages and utilities
- 🎯 Build caching and incremental builds
- 🚀 Fast CI/CD pipelines
- 📖 Shared documentation
- 🧪 Testing across packages

## Quick Start

\`\`\`bash
npx create-turbo@latest my-monorepo
cd my-monorepo
pnpm install
pnpm run build
\`\`\`
