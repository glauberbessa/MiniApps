# /scaffold-react-native

**React Native mobile app with Expo, navigation, and native modules.**

Use this skill when building cross-platform iOS and Android apps.

## What This Scaffold Provides

- 📱 Expo framework
- 🧭 React Navigation
- 🎨 NativeWind styling
- 🔐 Authentication setup
- 📡 API integration
- 💾 AsyncStorage
- 🧪 Testing setup

## Quick Start

\`\`\`bash
npx create-expo-app MyApp
cd MyApp
npm install expo-router nativewind zustand
npm start
\`\`\`
