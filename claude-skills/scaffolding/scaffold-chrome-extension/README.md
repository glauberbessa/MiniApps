# /scaffold-chrome-extension

**Chrome browser extension with React, message passing, and content scripts.**

Use this skill when building browser extensions with modern frontend tools.

## What This Scaffold Provides

- 🔌 Chrome Extension APIs
- ⚛️ React for UI
- 🔄 Message passing
- 🎨 Tailwind CSS styling
- 📦 Webpack bundling
- 🧪 Testing setup
- 🚀 Auto-update support

## Quick Start

\`\`\`bash
mkdir my-extension
cd my-extension
npm init -y
npm install webpack react react-dom tailwindcss
npm run build
\`\`\`
