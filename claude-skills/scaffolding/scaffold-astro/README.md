# /scaffold-astro

**Astro static/hybrid site with MDX content, dark mode, and SEO optimization.**

Use this skill when building fast static sites, blogs, or hybrid SSR applications.

## What This Scaffold Provides

- ⚡ Astro framework (static + SSR)
- 📝 MDX for content
- 🎨 Tailwind CSS styling
- 🌙 Dark mode support
- 📊 SEO optimization
- 🧪 Testing setup
- 📦 Content collections
- 🚀 Deployment ready

## Quick Start

\`\`\`bash
npm create astro@latest
cd my-site
npm install @astrojs/mdx
npm run dev
\`\`\`
