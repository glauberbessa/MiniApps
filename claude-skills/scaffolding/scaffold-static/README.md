# /scaffold-static

**Static site with Framer components, animations, and interactive elements.**

Use this skill when building fast, interactive static websites.

## What This Scaffold Provides

- 🎨 Framer for interactive components
- ✨ Smooth animations
- 🎯 Responsive design
- 📄 Static site generation
- 🚀 Zero JavaScript (optional)
- 📦 Easy deployment
- ♿ Accessible components

## Quick Start

\`\`\`bash
npm create vite@latest my-site -- --template react
npm install framer-motion
npm run dev
\`\`\`
